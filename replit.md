# EduGov Connect

A U.S. Government unified education platform — Stage 1.

## Tech Stack
- React 18 + TypeScript
- Vite 5
- Tailwind CSS v3
- React Router v6
- Lucide React icons

## Running
```bash
npm install
npm run dev
```
